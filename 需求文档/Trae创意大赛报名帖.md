# 【标题】生活娱乐 · 琴键飞舞 —— 用电脑键盘弹钢琴的 AI 沉浸式音乐游戏

---

# 【正文】

## 1. 创意名称 + 创意介绍

**创意名称：琴键飞舞（KeyDance Piano）**

**想解决什么问题：** 学钢琴门槛高、成本贵，普通人想随时随地弹一首曲子放松心情，却苦于没有乐器、不会识谱、找不到适合自己的曲谱。市面上的钢琴 App 要么是纯模拟弹奏缺乏引导，要么需要手动寻找 MIDI 文件，普通人根本无从下手。

**为什么会想到做这个：** 灵感来源于生活中常见的场景——很多人在听到一首喜欢的歌时，会忍不住在桌面上用手指"弹空气钢琴"，或者在键盘上随意敲击试图还原旋律。如果能把这个本能动作变成真正能出声、能打分、能学习的游戏体验，再配上 AI 自动把任意音频转成可弹的乐谱，就能让任何人零门槛享受弹琴的乐趣。TRAE AI 大赛的核心是"用 AI 解决实际问题"，而"让 AI 听懂音乐并自动写谱"正是把这个能力落到了每个人都能触碰的娱乐场景里。

**大概是什么产品：** 一款纯前端网页游戏（网站），打开浏览器即可玩，无需下载安装。用电脑键盘或手机触屏弹奏钢琴，支持跟弹打分和自由弹奏双模式，内置 AI 音频转谱功能（上传本地音频文件自动生成乐谱和按键提示），配 Three.js 3D 钢琴视觉、粒子特效、二次元风格背景与摄像头画中画，风格可由用户自由切换。零后端、零安装，单 HTML 压缩包即可提交体验。

---

## 2. 目标用户及痛点

**面向哪些用户：** 男女老少皆宜。核心用户群体包括三类——① 喜欢音乐但没学过钢琴的普通人（学生、上班族，想放松解压）；② 有一定钢琴基础、想用碎片时间练琴的爱好者；③ 对 AI 技术好奇、想体验"上传首歌就能自动生成乐谱"的科技尝鲜者。

**在什么场景下使用：**
- 午休或下班后，打开网页弹一首《天空之城》放松心情
- 听到一首喜欢的歌，上传音频让 AI 自动转谱，马上就能跟着弹
- 朋友聚会时用大屏投出来，大家轮流挑战跟弹打分，比谁连击更高
- 手机横屏也能玩，通勤路上戴上耳机弹几曲

**当前痛点：**
- **学琴门槛高：** 传统钢琴动辄上万元，线下课程按小时收费，识谱更需要长期练习，把大量想"随便弹弹"的人挡在门外
- **现有产品不够"傻瓜"：** Synthesia 等软件需要用户自己找 MIDI 文件导入，对普通人来说 MIDI 是什么都不清楚；Virtual Piano 只能自由弹奏没有引导和打分，弹得对不对全靠自己听
- **没有"听歌即弹"的能力：** 用户听到一首歌想弹，现有工具无法自动把音频变成可弹的乐谱，手动扒谱需要专业听力
- **缺乏沉浸感：** 多数网页钢琴就是一张静态琴键图，没有视觉氛围，弹两下就无聊了
- **移动端体验差：** 大部分键盘钢琴游戏只考虑桌面端，手机用户被排除在外

## 3. 价值与意义

**效率提升（降低音乐体验门槛）：**

琴键飞舞把"想弹一首歌"的全流程从"买琴→找老师→学识谱→练习数月"压缩到"打开网页→选曲→跟着按键提示弹"的几秒钟。AI 转谱功能更进一步：用户上传任意一段音频，Spotify Basic Pitch 神经网络在浏览器端自动识别音高和节奏，生成乐谱与下落式按键提示，让"听到就能弹"成为现实。这意味着用户不需要任何乐理知识，也不需要到处找 MIDI 文件，只需要一首自己喜欢的歌的音频文件，就能立刻开始弹奏。判定系统给出 Perfect/Great/Good/Miss 实时反馈和连击得分，历史记录追踪进步曲线，让练习有目标、有成就感。

**社会价值（音乐普惠与情绪疗愈）：**

音乐是人类最原始的情绪出口，但乐器演奏长期被"专业门槛"和"经济成本"绑定，成为少数人的特权。琴键飞舞用一张网页、一把键盘，把钢琴演奏的体验还给每一个人——不需要买琴、不需要报班、不需要识谱，老人可以弹一首《东方红》回忆青春，孩子可以弹《小星星》培养乐感，上班族可以弹《卡农》释放压力。摄像头画中画让用户看到自己弹琴的样子，3D 钢琴和粒子特效营造演奏厅般的沉浸氛围，把"弹琴"从技能学习变成纯粹的情绪享受。纯前端架构确保所有音频处理在本地完成，用户隐私零泄露，断网也能玩。

## 附：核心功能一览

| 功能模块 | 说明 |
|---|---|
| 音源引擎 | Web Audio API 低延迟发声，支持钢琴/音乐盒/Pad 多音色，按下到出声 < 50ms |
| 跟弹打分 | 下落音符 + 判定窗口（Perfect/Great/Good/Miss）+ 连击 + 结算评分，写入历史记录 |
| 自由弹奏 | 任意按键即时发声，支持八度切换（Z/X 键）、多音色切换、录音回放 |
| AI 转谱 | 上传 MP4/音频 → 浏览器端 Basic Pitch 推理 → 自动生成乐谱 + 按键提示，音频不出本机 |
| 乐谱展示 | OpenSheetMusicDisplay 渲染五线谱，跟弹时高亮当前小节，支持缩放/隐藏 |
| 沉浸视觉 | Three.js 3D 钢琴 + 粒子爆发特效 + 二次元/星空/赛博风格背景切换 + 摄像头画中画 |
| 内置曲库 | 8-12 首公有领域经典曲目（小星星/欢乐颂/天空之城/卡农等），按难度分类 |
| 歌单推荐 | 基于用户历史和偏好的规则推荐，"放松时刻/欢快节奏/经典入门"歌单 |
| 历史记录 | IndexedDB 本地持久化，记录得分曲线与录音，可导出 MIDI 文件 |
| 全平台适配 | 桌面键盘 + 手机横屏触屏琴键（≥44px），响应式自适应 |
| 新手引导 | 首次进入分步遮罩引导，常驻帮助入口，首曲自动进入"等待模式" |
| 无障碍 | 高对比度模式、键盘全可达、振动反馈、色弱友好配色 |

> **技术亮点：** 全部功能纯前端实现（Web Audio API + Spotify Basic Pitch + Three.js + OpenSheetMusicDisplay），零后端、零安装，单 HTML 压缩包提交即体验。AI 转谱模型浏览器端懒加载，音频处理全程本地完成，隐私友好，断网可用。
